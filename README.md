🏫 經費領據雲端管理系統 (Google實名安全版 - V9.0 Final)



這是一套專為學校、公務機關、教育機構設計的「線上領據收集與智能排版系統」。

完全免安裝、免記密碼，支援手機電子簽名與存摺一鍵上傳，並具備軍規等級的資料加密防護 (E2EE)。



無論您是資訊組長還是非資訊專長的行政老師，都能 0 秒無痛上手，立即優化您的行政流程！



🚀 立即體驗 (完全免費、免部署)



您不需要具備任何寫程式的背景，也不需要懂如何架設伺服器。請直接點擊下方網址，登入您的 Google 帳號即可開始使用：



👉 進入系統：https://movenorthward.github.io/school-receipt/



【如何運作？只要 3 個步驟】



點擊進入後，選擇「⚙️ 行政管理員」並用您的 Google 帳號實名登入。



系統會在背景為您自動建立一個 100% 獨立的專屬頻道與安全金鑰。



點擊後台的「🔗 複製專屬講師連結」傳給講師，講師填寫完的資料只會傳送到您的專屬後台。



🔐 核心防護：真正的端到端加密 (E2EE)



本系統將講師的隱私放在第一位。我們導入了通常只在通訊軟體（如 WhatsApp、Signal）才見得到的 端到端非對稱加密 (E2EE - End-to-End Encryption) 技術：



資料全自動上鎖：講師在手機上按下送出的瞬間，系統會使用 AES-256 演算法，直接在講師的手機端將「身分證、存摺、簽名、電話」加密成一堆無意義的亂碼。



只有承辦人能解開：解開這堆亂碼的「私鑰」，是綁定在承辦人（您）的 Google 帳號上的。



開發者也看嘸：即使是系統開發者、資料庫管理員，甚至是駭客，如果沒有您的 Google 帳號登入授權，在資料庫中看到的永遠只是一堆無法破解的亂碼！



✨ V9.0 系統獨家亮點功能



1\. 完美 A4 PDF 排版直出 (絕不跑版)



內建獨家「隱形畫布引擎」。傳統網頁列印常遇到表格被切斷、文字跑版的問題，本系統能將領據與圖檔轉換為完美的 A4 PDF，按下「🖨️ 產生 PDF 並列印」，保證 100% 精準對齊。



2\. 浮空簽名檔與原生去背



講師的手寫簽名會自動存為 原生透明 PNG。在排版預覽時，承辦人可以直接用滑鼠點擊簽名，彈出視窗自由拖曳位置與縮放大小，簽名會完美浮空覆蓋於表格線上，再也不會出現醜陋的白底方塊！



3\. 動態分流與 Session 記憶防呆



同一個網址，不同老師登入，會自動分流到完全獨立的專屬資料夾，跨處室共用也絕不打架。此外，系統內建記憶路由，完美解決了講師用手機點擊 LINE 連結跳轉登入後，網址遺失導致的錯誤。



4\. 批次處理與一鍵打包匯出



支援勾選多位講師進行「批次排版列印」。核銷時，可一鍵將清單匯出為 Excel 名冊，並將所有簽名與存摺圖片打包成 ZIP 檔一次下載。



🛠️ 給資訊組長與開發者的話 (Fork \& Deploy)



本系統採用純前端架構 (React + Tailwind CSS)，無須架設後端伺服器 (Serverless)。如果您希望將系統部署在自己學校的網域下：



將此專案 Fork 到您的 GitHub。



前往 Google Firebase 申請一個免費的專案與 Firestore Database。



將 Firebase 提供的 firebaseConfig 貼入 index.html 的第 47 行。



啟用 Firebase Authentication 的「Google 登入」與「匿名登入」功能。



部署至 GitHub Pages，您的學校專屬領據系統就上線了！
