🏫 經費領據雲端管理系統 (Google實名安全版)

這是一套專為學校、公務機關、教育機構設計的「線上領據收集與排版系統」。
完全免安裝、免記密碼，支援手機電子簽名與存摺上傳，並具備最高等級的資料加密防護。

非資訊專長的老師也能 0 秒無痛上手，立即開始收集講師領據！

🚀 立即體驗 (完全免費、免部署)

您不需要具備任何寫程式的背景，也不需要懂如何架設伺服器。我們提供兩種最簡單的使用方式：

方案一：直接使用發布版網址 (最推薦 🌟)

請直接點擊下方網址，登入您的 Google 帳號即可開始使用：

👉 進入系統：https://movenorthward.github.io/school-receipt/

【如何運作？】

點擊進入後，選擇「⚙️ 行政管理員」並用您的 Google 帳號登入。

系統會在背景為您自動建立一個 100% 獨立的專屬頻道。

點擊「🔗 複製專屬講師連結」傳給講師，講師填寫完的資料只會傳送到您的專屬後台。

別人絕對看不到您的資料，您也看不到別人的資料！

方案二：使用 AI Canvas 模式 (進階自訂)

如果您希望修改領據上的欄位、文字，或是想體驗最新的 AI 協作開發：

複製本專案中的 app.jsx (或 index.html) 原始碼。

貼到支援 Canvas 模式的 AI 平台（例如 Google Gemini）。

AI 會直接在右側渲染出系統畫面，您可以在不架設任何網站的情況下，直接在 AI 視窗中操作、收集資料與列印！

🔐 核心特色：軍規級的資料安全 (E2EE)

本系統使用 Firebase 作為雲端資料庫，但您完全不需要擔心講師個資外洩給系統架設者！

本系統導入了 端到端非對稱加密 (E2EE - End-to-End Encryption) 技術：

資料全自動上鎖：講師在手機上按下送出前，系統會先使用 AES-256 在講師的手機端將「身分證、存摺、簽名、電話」加密成一堆無意義的亂碼。

只有您能解開：解開這堆亂碼的「私鑰」，是綁定在承辦人（您）的 Google 帳號上的。

真正的隱私隔離：即使是開發者、資料庫管理員，甚至是駭客，如果沒有您的 Google 帳號登入授權，在資料庫後台看到的永遠只是一堆亂碼，絕對無法得知任何講師的真實資訊！

✨ 系統亮點功能

Google 實名認證：全面綁定 Google 帳號，一鍵登入免記密碼，阻擋惡意機器人。

完美 A4 PDF 排版直出：內建隱形畫布引擎，無論是列印領據、存摺封面還是電子簽名，按下「🖨️ 產生 PDF 並列印」保證 100% 精準對齊、絕對不跑版。

動態分流空間：同一個網址，不同老師登入，會自動分流到完全獨立的專屬資料夾，跨處室共用也絕不打架。

防呆提示與一鍵匯出：內建 Excel 單鍵匯出與 ZIP 打包下載（含圖片檔），方便後續核銷作業。

🛠️ 給開發者的話 (Fork & Deploy)

如果您是學校的資訊組長，想要將系統部署在自己學校的網域下：

將此專案 Fork 到您的 GitHub。

前往 Firebase 申請一個免費的專案與 Firestore Database。

將 Firebase 提供的 firebaseConfig 貼入 index.html 的第 36 行。

部署至 GitHub Pages，您的學校專屬領據系統就上線了！
